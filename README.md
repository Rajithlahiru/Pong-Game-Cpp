# Pong game 

## Using C++ and Raylib

### Step to create

* Create a blank screen & the Game Loop
* Draw the paddles & the ball
* Move the ball around
* Check for collisions with the edges
* Move the player paddle
* Move the CPU paddle with AI
* Check for collisions
* Add scoring
* Add colors
